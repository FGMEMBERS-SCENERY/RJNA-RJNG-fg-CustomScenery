**********************************
    インストールの手順
**********************************

①事前に地形データをインストールしておく必要があります。
　FlightGearサイトより[e130n30]のシーナリーをインストール。
　※シーナリーのインストール方法はFlightGearサイト参照

②解凍したフォルダ内にある[Airport-RJNA]フォルダを
　FlightGearフォルダ内の\data\Modelsの直下に移動。

③解凍したフォルダ内にある[e136n35]フォルダから
　[5185355.btg.gz]ファイル
　[5185363.btg.gz]ファイル
　[5185363.stg]ファイル
　[RJNA.btg.gz]ファイル
　をコピーして、シーナリーの保存先
  \scenery\Terrain\e130n30\e136n35内の同名ファイルと置換え
　してください。既に他のモデルを配置している場合は
　[5185363.stg]ファイルをテキストエディタ等で開き、
　配置情報をコピーして既存ファイルへ追加してください。

　注１．FG SCENERYの優先順位によって表示されない場合があります。
　　　　優先順位を変更するか、優先している側へもコピーするなど
　　　　対応してください。
　注２．空港周辺の地形データはカスタム作成したもののためオリジナルの
　　　　地形とズレが生じます。
